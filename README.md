# minipv

## 小程序简介
该项目使用uni-app进行开发，通过vue-cli创建项目，集图片、视频、音乐功能于一身的多元化小程序，其中后台接口使用了图片获取和网易云音乐node.js版接口。

## 功能实现

- 图片
  - [x] 浏览
  - [x] 下载
  - [x] 专辑分类
  - [x] 评论浏览
- 视频
  - [x] 播放、暂停
  - [x] 下载
  - [x] 分享
  - [x] 静音
  - [x] 发送弹幕

- 音乐
  - [x] 歌单
  - [x] 搜索
  - [x] 音乐播放
  - [x] banner

## Build SetUp

```
# 下载或克隆下来，然后安装依赖
npm install

# 开发预览
npm run dev:mp-weixin

# 具体查看开发文档
```
